Purpose:

  • Display Login activites of different users with time
  • Compare Login interest of different users!
  • View the login count of a particular user on a particular day
Analytics

From the graph it can be clearly inferred the login activities of different users, which can be used to analyze which users are most active on the page. We can also select/deselect a user and compare the login activity of a particular user with all the remaining users.

Interactions
  • Hover: Display the login count of a particular user at pariculat point in time.
  • click: Clicking on usernames would display the data of only the selected users login with respect to time. We can hide select login pattern of any user
Steps to find results
  • From the chart it is clear which user has most login on a particular date
  • On Hovering the graph any node in the graph we can note the number of times that user has logged in on a particular date
  • Click on the username in the graph to view only the pattern of few users.
  • Enabale/Disable a particular user to view his login activity versus all the other users.
Why does it matter?
  • To compare how frequently the user logs in to the application
  • To understand the pattern of variation of login among the users.
  • To compare users login activity with other users for predictive modeling.
