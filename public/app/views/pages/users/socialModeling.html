User Operations frequency distribution








Purpose:

  • Display Frequency Distribution of actions of logger User
  • Display Frequency Distribution of actions of all users
  • Display Frequency Distribution of specific actions
  • Compare Frequency Distribution of different actions and users
Analytics

From The pie chart it can be inferred what action a particular user was mostly engaged in. The chart tells us about the user action which mostly lies in comments, sharing, searching a particular question, upvoting a question as well as downvoting a question. The frequency of each action is mentioned in terms of percentage. This tells us about the pattern of frequency of actions for all the users of the application. We can analyse different actions by clicking on them which will enable/disable the result of that action on the pie chart.

Interactions
  • Hover: Display the frequency in terms of percentage of a particular user for a particular action on stack overflow page
  • click: when we click on the action it hides the graph of that action to analyze individual action or other actions.
  • Button to View/Hide all user frequency distribution
Steps to find results
  • Extracted the action data for a particular user as well as all the users from db.
  • Click on Button to view/hide all user frequency distribution
  • From the chart it is clear which action is performed most by the user.
  • Oon clicking on the action we can select which action to view in the distribution.
Why does it matter?
  • To know which action is most common among the users which can be user for user modelling.
  • To understand the pattern of variation of actions among the users.
