Why do you decide to log these actions?

These logs tell us about the user actions and behaviour on stack overflow page. For instance: click, voteup, vote down, share, scroll etc. These logs can be used to analyse user's point of interest and time spent on different questions. The actions are decomposed as follows:

  • Upvoted : Event triggered when user upvotes an answer
  • Devoted : Event triggered when User Downvotes the answer
  • comments: Event triggered when user clicks on add comment
  • Share : Event triggered when user clicks on share answer
  • Search : Event triggered when user clicks on search field
  • Scroll : Event triggered when user scrolls to the end of the page

    • These logs can be used to recommend user his favourite questions on top or any other similar suggestions.
